# VidJot

## Description
Storybooks is an example of a video idea publishing app using Node.js, Express, and MongoDB with Bootstrap CSS.  It uses local database authentication, and Handlebars to compile templates into JavaScript functions.

## Installation

- Install Node.js and create a MongoDB instance 
- Clone the repo
    ```
    git clone https://github.com/phil-stt/vidjot
    ```
- Install node packages `npm install`
- Create a settings file `env.development` with the following parameters updated to your environment:
    ```
    PORT=5700
    DATABASE_URL=mongodb://localhost/vidjot-dev
    ```
- Run `npm start-dev` to start the app with development parameters


## Usage

1.  Access the application in your browser at: `http://localhost:5700`
2.  Click Register to create a VidJot account and login
3.  Add your video idea!
